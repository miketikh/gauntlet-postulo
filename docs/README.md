## Quick Reference Cards

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold |
| `Cmd/Ctrl + I` | Italic |
| `Cmd/Ctrl + U` | Underline |
| `Cmd/Ctrl + K` | Insert link |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Esc` | Cancel refinement |
| `Cmd/Ctrl + Enter` | Apply refinement |

### Quick Actions (AI Refinement)

1. **Make More Assertive** - Strengthen demands
2. **Add More Detail** - Expand with context
3. **Shorten This Section** - Condense content
4. **Emphasize Liability** - Highlight defendant fault
5. **Soften Tone** - Make conciliatory
6. **Improve Clarity** - Simplify language

### Common File Formats

**Upload**:
- PDF: Police reports, medical records
- DOCX: Word documents
- JPG/PNG: Photos, scanned documents

**Export**:
- DOCX: Microsoft Word format
- PDF: Via Word (export to Word, then Save As PDF)

## About This Documentation

### What's Included

This documentation suite covers:
- **All User Features**: Complete guides for end users
- **Administrative Functions**: Firm management and settings
- **Developer Resources**: API, architecture, setup
- **Troubleshooting**: Common issues and solutions
- **Tutorials**: Hands-on learning paths
- **Reference**: Quick lookups and FAQs

### What's Not Included

**Not Covered**:
- Firm-specific workflows (consult your admin)
- Legal advice or strategy (Steno is a tool, not counsel)
- Advanced customization (contact support)
- Upcoming features (see FAQ roadmap section)

### Documentation Philosophy

**User-Focused**:
- Written for attorneys, paralegals, admins, developers
- Clear, simple language (no unnecessary jargon)
- Practical examples and use cases
- Step-by-step instructions
- Scannable formatting

**Comprehensive**:
- Covers all features implemented in Wave 1 & 2

**Maintainable**:
