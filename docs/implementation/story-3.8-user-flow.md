## Mobile Responsiveness

All dialogs and modals are responsive:
- **Desktop:** Full-width modals with side-by-side actions
- **Tablet:** Stacked layout with readable content
- **Mobile:** Full-screen dialogs with scrollable content

---

## Keyboard Navigation

All dialogs support standard keyboard shortcuts:
- `ESC` - Close dialog
- `Tab` / `Shift+Tab` - Navigate between buttons
- `Enter` - Activate focused button
- `Space` - Activate focused button

---

## Visual States

### Loading
- Spinner animation
- "Loading version history..." message
- Disabled action buttons

### Success
- Green checkmark icon
- Success message
- Auto-close after 1.5 seconds

### Error
- Red error icon
- Clear error message
- Retry or close options

---

## Design Principles

1. **Clear Visual Hierarchy:** Version numbers prominent, metadata secondary
2. **Consistent Layout:** All components use same design language
3. **Defensive Actions:** Confirmation required before destructive operations
4. **Helpful Guidance:** Explanations for all actions
5. **Graceful Degradation:** Clear error messages and recovery options
6. **Accessibility:** WCAG compliant with keyboard support
7. **Responsive:** Works on all screen sizes
